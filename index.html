PETUNJUK UJIAN :
  1. Buka Link yang dibagikan oleh Pengawas/Walikelas
  2. Tekan Tombol Absen sesuai Kelas
  3. Isi Biodata dengan lengkap dan benar, kemudian klik tombol Kirim / Submit
  4. Setelah menekan tombol Kirim / Submit, muncul TOKEN SOAL dan LINK SOAL.
    Silahkan catat TOKEN SOAL tersebut kemudian buka LINK SOAL
  5. Pada Halaman LINK SOAL, Klik Tombol "Mulai" pada mata pelajaran yang akan diujiankan
  6. Isi Biodata dan masukkan TOKEN SOAL yang muncul setelah mengisi Absen
  7. silahkan mengerjakan Soal yang muncul
  8. Setelah mengerjakan soal, pastikan untuk mengklik Tombol Kirim/Submit
  9. JAWABAN TIDAK AKAN TERKIRIM, JIKA TIDAK MENGKLIK TOMBOL KIRIM/SUBMIT
